# Learning Tracker

A premium, collaborative web application to track your daily progress and learning goals.

## Features

- **Personalized Tracking**: Set daily goals for various subjects and tasks.
- **Consistency Matrix**: A GitHub-style matrix to visualize your progress over time.
- **Collaborative Rooms**: Join rooms with friends to stay motivated and see their streaks.
- **AI-Powered Tips**: Get high-impact daily tips generated by Gemini AI tailored for your learning path.
- **Admin Dashboard**: Manage users and logs through a secure control center.
- **Persistence**: Your progress is saved in a local SQLite database.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- Gemini API Key (for AI tips)

### Installation

1. Clone the repository and navigate to the directory.
2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
3. Create a `.env` file based on `.env.example`:
   ```bash
   cp .env.example .env
   ```
4. Add your `GEMINI_API_KEY` to the `.env` file.

### Running Locally

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deployment

### Docker

You can use the provided `Dockerfile` to deploy as a container:

```bash
docker build -t de-catalyst .
docker run -p 3000:3000 --env-file .env de-catalyst
```

### Cloud Providers

- **Google Cloud Run**: Ideal for this containerized app.
- **Render / Railway**: Simple deployment by connecting your GitHub repo.

## Tech Stack

- **Frontend**: React 18, Tailwind CSS 3, Framer Motion, Lucide Icons, Konva (Canvas).
- **Backend**: Express (Node.js), WebSocket (ws), SQLite (better-sqlite3).
- **AI**: Google Gemini AI (gemini-1.5-flash).
